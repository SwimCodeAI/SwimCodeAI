  # Swim Code                                                                                              
                                                                                                           
  **AI-native Kanban IDE that runs on your desktop.**                                                      
   
  Cards flow through a visual pipeline — Backlog, Plan, In Progress, Test,                                 
  Commit, Done — and autonomous AI agents process each stage in isolated
  git worktrees. No cloud. No telemetry. Your code never leaves your                                       
  machine.                                                                                                 
                                                                                                           
  ## How it works                                                                                          
                                                        
  1. Drop a card in Backlog with a short description of what you want                                      
  2. A planner agent reads your codebase and writes an implementation plan
  3. A developer agent implements it in an isolated git worktree                                           
  4. A tester agent runs your test suite, iterates until it passes                                         
  5. A committer agent merges the branch when everything is clean                                          
  6. You review the diff at any point — human-review lane is optional                                      
                                                                                                           
  Cards flow automatically from stage to stage. You watch, intervene, or                                   
  let it run.

  ## Privacy                                            

  Swim Code runs entirely on your machine. There is no backend server                                      
  collecting your code, prompts, or activity.
                                                                                                           
  - [Privacy policy](https://swimcode.ai/privacy)                                                          
  - [Terms](https://swimcode.ai/terms)                                                                     
                                                                                                           
  The Swim Code GitHub App is how the desktop app signs you in and browses                                 
  your repositories. Access tokens are AES-256 encrypted in a local
  database and never leave your device. Revoke anytime via                                                 
  [github.com/settings/apps/authorizations](https://github.com/settings/apps/authorizations).
